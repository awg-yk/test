# 観測所ファインダー（Weather Station Finder）

気象庁の「過去の気象データ・ダウンロード」を、研究・解析向けに使いやすくするための非公式ツールです。
気象庁サイトの置き換えではなく、公開データをより効率よく取得するための補助ツールを目指します。

## 現在の状態: フェーズ1（データ基盤 + 一覧表示）

- 観測所マスタデータ（サンプル：全47都道府県の代表地点）を読み込み、一覧表示します。
- 地域一括選択・観測要素フィルタは未実装です（UIにプレースホルダーとして表示）。

## ローカルでの動作確認

`index.html` を直接ダブルクリックで開くと、ブラウザのセキュリティ制限（CORS）により
`fetch("data/stations.json")` が失敗します。必ず簡易サーバー経由で開いてください。

```bash
cd weather-station-finder
python3 -m http.server 8000
# その後ブラウザで http://localhost:8000 を開く
```

Node.js がある場合は `npx serve` でも構いません。

## データについて（重要）

`data/stations.json` は **プロトタイプ用のサンプルデータ** です。
各都道府県の代表的な観測地点を1件ずつ、大まかな座標・観測要素で収録していますが、
実運用前に気象庁の最新の観測所一覧と照合し、正確なデータに差し替える必要があります。

このツールは気象庁の生観測データを再配布するものではなく、観測所を探すための
メタデータ（地点名・地方・都道府県・観測要素など）のみを扱う設計です。
実際の気象データ取得は、最終的に気象庁の該当ページへ案内する形を想定しています。
